Security Context로부터 얻어낸 UserDetails 객체(사용자정보 객체(Member)를 품은 객체)를 Model로 받아보기

username: [[${username}]]

name: [[${name}]]

role: [[${role}]]

Principal 객체 값을 Model로 받기 : 사용자 인증 정보를 가진 객체

username2: [[${username2}]]

@AuthenticationPrincipal을 사용하여 UserDetails의 구현체(CustomUserDetails)를 파라미터로 주입받아 사용하기

username3 :[[${username2}]]

name3 :[[${name3}]]

role3 :[[${role3}]]
